## About this project

![Badge](https://img.shields.io/badge/to--do-application-brightgreen)

This is a to-do-list web application, where one can add his/her todos.

## Technologies Used

 - ReactJS
 - CSS
 - React-Bootstrap
 - Localstorage (to store the todos)
